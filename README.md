# Sistema-de-biofeedback-para-el-control-del-estr-s-ECG-EEG-
Una interfaz gráfica que permite cargar los .acq que contienen señales eeg y ecg obtenidos mediante el Biopac MP36 y poder analizarlos en ventanas de 2 segundos, permitiendo el generar un semáforo que nos indique si estamos en un estado de reposo (ojos cerrados) o un estado de estrés (ojos abiertos).
